Laravel Quick Notes

To get started (especially for the old project to have it run again on your machine), you need to have ‘composer‘ installed, then you should be able to enjoy by following below steps:

  1. composer create-project –prefer-dist laravel/laravel blog
  2. sudo chmod -R guo+w storage (allow write to storage folder)
  3. php artisan storage:link; php artisan cache:clear (need to link to public folder and for some old project, need clear cache)
  4. php artisan serve (start built-in web server)
  5. composer require laravel/ui
  6. php artisan ui bootstrap –auth (this for laravel 6.x, but for 5.x, use ‘php artisan make:auth‘)
  7. npm install && npm run dev
  8. php artisan migrate (not required for fresh start, check .env for db config before run this)
  9. now check http://localhost:8000 if we can see login/register at the top right

Next step you might want to run host virtual-host so that can access many projects at the same time using friendly name/domain like http://firstproject.dev from local machine. I recommend to use Xampp, it’s opensouce and easy to configure :)

Quick tips to move laravel from local machine to server

*Important things: Export DB, remove 'node_modules' folder, might use rsync or scp to copy project to server

While copy project to server like to '/var/www/laravel', you need to configure your apache2 or httpd server to point to this folder so that we can access via that IP address or name.

Then to make it work properly, do this:

  • Run: 'npm install --no-dev'
  • Delete 'public folder' and relink: 'php artisan storage:link'
  • Permission and ownership: 'udo chown -R :www-data laravel; sudo chmod -R gu+w storage; sudo chmod -R 775 bootstrap/cache; php artisan cache:clear'

Quick and easy use AdminLTE with Laravel

Follow this easy step:

  1. composer require jeroennoten/laravel-adminlte
  2. php artisan adminlte:install
  3. composer create-project laravel/laravel adminLte60 6.0.*

We can run by move into directory of project then issue:

  • php artisan serve

and now we can access project in the browser at http://localhost:8000

You can also use 'php artisan --host YOUR_IP_Addr --port SOME_PORT' to host your server for local network or else.
